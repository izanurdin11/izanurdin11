CARA INSTALL SERVER Minecraft

Langkah 1: Persiapan

Unduh dan instal Java. Minecraft server memerlukan Java agar dapat berjalan. Pastikan Anda memiliki versi terbaru dari Java.

Langkah 2: Unduh Minecraft Server

Kunjungi situs web pepermc.io dan unduh file peper.jar Simpan file tersebut di folder yang mudah diakses.

Langkah 3: Mulai Server

Buka terminal atau command prompt, navigasi ke folder tempat Anda menyimpan file server, dan jalankan perintah berikut:

java -Xmx1024M -Xms1024M -jar minecraft_server.jar nogui

Langkah 4: Konfigurasi Server

Setelah server dijalankan pertama kali, beberapa file konfigurasi akan dibuat. Buka file server.properties untuk mengatur preferensi server Anda, seperti mode permainan, maksimal pemain, dan lain-lain.

Langkah 5: Port Forwarding

Jika Anda ingin pemain dari luar jaringan lokal Anda bergabung, Anda perlu melakukan port forwarding di router Anda. Forward port 25565 ke alamat IP lokal komputer yang menjalankan server Minecraft.

Langkah 6: Mulai Bermain

Ajak teman-teman Anda untuk bergabung dengan memasukkan alamat IP publik Anda ke dalam Minecraft. Selamat bermain!
